# Förstå responsiv layout {#understanding-responsive-layout-with-aem-sites}

{{edge-delivery-services-and-page-editor}}

Med Adobe Experience Manager Sites kan en författare hantera bredden på en komponent baserat på enhetens bredd för att underlätta en responsiv webbplatsdesign. Med layoutläget och layoutbehållaren kan användarna skapa responsivt innehåll för olika enheter och förhandsgranska slutanvändarens upplevelse, utan att behöva lämna användargränssnittet i AEM.
