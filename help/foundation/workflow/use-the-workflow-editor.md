# Använda arbetsflödesredigeraren

Arbetsflödet möjliggör hantering av affärsprocesser i Experience Manager och används för automatisk bearbetning av innehåll samt underlättar styrning och processer som kräver mänsklig beslutsfattande. Arbetsflödet definieras via arbetsflödesmodeller som består av en serie steg och skapas och hanteras i [!UICONTROL Workflow Model Editor].
