# Konfigurera AEM Assets Content Hub

Lär dig hur du konfigurerar Experience Manager Assets Content Hub med Adobe Cloud Manager och resurstillägg. Lär dig även hur du konfigurerar åtkomst för användare med produktprofiler.

## Konfigurera AEM Assets Dynamic Media OpenAPI:er

Om du har licens för AEM Assets Dynamic Media OpenAPI:er kan du använda dem för att generera dynamiska renderingar i Content Hub, till exempel Smart Crops. Innan dessa Dynamic Media OpenAPI-återgivningar kan visas i Content Hub krävs en första konfiguration.
